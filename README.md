# What's the Weather

## Description

Find out the weather for a location by searching the city name and be presented with current and future weather conditions. The city will be added to the search history for ease of future searches. 

## Built with
* HTML
* CSS
* Javascript
* Bootstrap
* jQuery
* Moment.js
